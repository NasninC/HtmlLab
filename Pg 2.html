  2. Roman numbers
  3. Capital letters
  4. Small letters

Unordered List

Unordered list or bulleted list displays elements in unordered format.Ul tag is used for ordered list.


  • Disc
  • Circle
  • Square
  • None

Description list is a list used to define items in a specific way
